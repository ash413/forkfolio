# RecipeHub - Recipe Sharing Platform

## 📝 Project Overview
RecipeHub is a full-stack web application that allows users to share, discover, and interact with recipes. Users can create, view, like, and search for recipes from a community of home cooks and food enthusiasts.

## 🚀 Features
- User Authentication
  - Register and Login
  - Profile Management
- Recipe Management
  - Create new recipes
  - Edit and delete personal recipes
  - View recipe details
- Social Interactions
  - Like recipes
  - Search and filter recipes

## 🛠 Tech Stack
### Frontend
- React
- Tailwind CSS
- React Router
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

### Additional Tools
- Bcrypt (Password Hashing)
- Dotenv (Environment Variables)

## 📦 Prerequisites
- Node.js (v16+ recommended)
- MongoDB
- npm or yarn
